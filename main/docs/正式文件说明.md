# 东海帝王AI - 正式文件说明

## 📁 项目结构总览

```
agent/
├── start.bat               # 一键启动脚本（setup.bat部署完成后才能正常使用）
└── main/
    ├── main.py             # 程序入口
    ├── requirements.txt    # Python 依赖
    ├── setup.bat           # 环境安装脚本
    ├── assets/             # 静态资源
    ├── character/          # 角色相关资源
    ├── core/               # 核心功能模块
    ├── ui/                 # UI 界面模块
    ├── mcp/                # MCP 工具服务
    ├── memory/             # 记忆系统模块
    ├── scripts/            # 开发脚本
    ├── plugins/            # 外部插件
    ├── docs/               # 文档
    ├── data/               # 数据目录
    ├── logs/               # 日志目录
    └── 旧版本代码/         # 归档的旧版代码
```

---

## 🚀 入口文件

| 文件 | 说明 |
|------|------|
| **main.py** | 程序入口，初始化 PyQt5 应用、启动 HTTP 服务器（port 5000）、预加载向量模型 |

---

## 🧠 核心模块（core/）

| 文件 | 说明 |
|------|------|
| **improved_ai_agent.py** | AI Agent 核心逻辑，整合 LLM、记忆、工具、TTS、多模态 |
| **config.py** | 配置管理，负责加载和保存 `ai_agent_config.json` |
| **tool_manager.py** | 工具管理器，实现模块化工具注册和调用 |
| **utils.py** | 通用工具函数，系统辅助功能 |
| **gpt_sovits_unified.py** | 统一 TTS 调用模块，支持 Gradio 和 api_v2 两种方式 |
| **api_v2_client.py** | GPT-SoVITS API v2 客户端，支持流式和非流式合成 |
| **text_queue_manager.py** | 流式文本队列管理，按标点切分文本并顺序发送 TTS |
| **realtimestt_process.py** | 实时语音识别集成，将识别结果 POST 到主程序 |
| **realtime_stt_direct.py** | RealtimeSTT 直接集成模块 |
| **asr_integration.py** | ASR 集成模块，对接 RealtimeSTT 与主程序 |
| **godot_pet_client.py** | Godot 桌宠 TCP 客户端，控制桌宠动作/表情/状态 |
| **improved_ocr.py** | OCR 功能模块 |

---

## 🖥️ UI 模块（ui/）

| 文件 | 说明 |
|------|------|
| **main_window.py** | 主窗口，包含对话展示、录音按钮、模式切换等 UI 逻辑 |
| **settings_dialog.py** | 设置对话框（浅色主题） |
| **ui_dialogs.py** | 其他 UI 对话框，含记忆系统、MCP 工具等面板 |

---

## 🔌 MCP 模块（mcp/）

| 文件 | 说明 |
|------|------|
| **mcp_server.py** | MCP 服务端，注册和对外暴露工具 |
| **mcp_client.py** | MCP 本地客户端，调用 MCP 工具 |
| **tools/music.py** | 点歌工具，支持 QQ 音乐和网易云音乐（占位实现，待接入 API） |

---

## 🧬 记忆系统（memory/）

| 文件 | 说明 |
|------|------|
| **improved_memory.py** | 改进的记忆系统，使用 FAISS + SQLite 向量数据库，嵌入模型 `BAAI/bge-small-zh-v1.5` |

## 🚀 脚本（scripts/）

开发调试用脚本，正式启动请使用根目录的 `start.bat`。

| 文件 | 说明 |
|------|------|
| **start_all.bat** | 开发用：单独启动主程序 |
| **run_example.bat** | 开发用：单独启动 STT 集成程序 |
| **start_godot_pet.bat** | 启动 Godot 桌宠 |
| **download_models.py** | 下载所需模型 |
| **download_godot.py** | 下载 Godot 引擎 |

---

## 🔧 外部插件（plugins/）

| 目录 | 说明 |
|------|------|
| **plugins/GPT-SoVITS/** | GPT-SoVITS TTS 服务（独立启动，port 9880） |
| **plugins/Multimodal/** | 多模态处理插件（截图 + GLM-4V 视觉模型） |
| **plugins/godot/tokai-teio/** | Godot 桌宠项目源码 |

> RealtimeSTT 已通过 venv 安装为 Python 库，不再依赖 plugins/ 下的目录。

---

## 📄 配置文件

| 文件 | 说明 |
|------|------|
| **ai_agent_config.json** | 主配置文件（API 密钥、TTS 参数、功能开关等） |
| **ai_agent_config.example.json** | 配置文件示例（用于初始化，不含敏感信息） |

---

## 💡 启动说明

```bash
# 正式启动（根目录，一键启动 TTS → 主程序 → STT）
start.bat

# 单独启动 Godot 桌宠（可选）
main/scripts/start_godot_pet.bat
```
