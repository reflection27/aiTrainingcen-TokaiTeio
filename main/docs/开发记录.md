# 东海帝王AI担当 - 开发记录

## 2026-04-08

### 游戏画面监控系统（新增）

**背景**
多模态模式（per-message 截图）延迟高、性价比低，改为后台定时截图+缓存注入方案。

**新增文件**
- `plugins/Multimodal/game/umamusume/config.json`
- `plugins/Multimodal/game/minecraft/config.json`
- `plugins/Multimodal/game/云顶之弈/config.json`

**修改文件**

`plugins/Multimodal/screen_capture.py`
- 新增 `list_windows()` — 列出所有可见窗口标题（调试用）
- 新增 `_find_window_hwnd(title_keyword)` — 按标题关键词查找窗口句柄
- 新增 `capture_window_by_title(title_keyword)` — 按窗口标题截图（部分匹配，大小写不敏感）
- 依赖新增：`pywin32`（requirements.txt 已更新）

`plugins/Multimodal/multimodal_processor.py`
- 新增字段：`game_state_cache`、`_prev_screenshot_array`、`_monitor_thread`、`_monitor_stop_event`
- 新增 `list_game_configs()` — 扫描 game/ 目录返回可用游戏名
- 新增 `load_game_config(game_name)` — 从 config.json 加载游戏监控参数
- 新增 `start_game_monitoring_from_config(game_name)` — 按游戏名一键启动监控
- 新增 `start_game_monitoring(...)` — 手动参数启动监控，新增 `describe_prompt` 参数
- 新增 `stop_game_monitoring()` — 停止监控并清空缓存
- 新增 `_capture_for_monitoring(...)` — 监控专用截图（含resize）
- 新增 `_monitoring_loop(...)` — 后台监控主循环（diff检测 + GLM-4V描述 + cache更新）

`improved_ai_agent.py`
- `_get_system_prompt()` 改为动态注入 `game_state_cache`（游戏画面缓存）
- `_build_prompt()` 改用 `_get_system_prompt()` 替代直接引用 `self.system_prompt`
- `_process_request()` 新增 `game_monitoring_active` 判断，游戏监控运行时跳过 per-message 截图分支

**三种视觉模式（互斥）**

| 模式 | multimodal_enabled | 游戏监控 | 延迟 |
|------|-------------------|---------|------|
| 纯文本 | False | 停止 | 最低 |
| per-message 多模态 | True | 停止 | 高（每条消息调用GLM-4V） |
| 游戏陪玩 | False | 启动 | 低（缓存注入，零额外API调用） |

**各游戏监控参数**

| 游戏 | 截图方式 | 间隔 | diff阈值 | 分辨率 |
|------|---------|------|---------|-------|
| 赛马娘(umamusume) | 窗口截图 | 30s | 0.04 | 800×600 |
| Minecraft | 窗口截图 | 15s | 0.09 | 960×540 |
| 云顶之弈 | 全屏截图 | 10s | 0.10 | 1024×576 |

**使用方式**
```python
# 启动游戏监控
agent.multimodal_processor.start_game_monitoring_from_config("umamusume")

# 查看可用配置
MultimodalProcessor.list_game_configs()

# 停止
agent.multimodal_processor.stop_game_monitoring()
```

### 桌宠功能（进行中）

**技术选型**：Godot 4.6.2 + 3D MMD 模型（.pmx）

**模型信息**
- 角色：东海帝王（トウカイテイオー）
- 来源：UmaViewer 2.1.8 提取，模型ID 1003
- 骨骼：413根，含完整面部/手指/耳朵/尾巴骨骼
- 表情Morph：240个（眼/眉/嘴/耳）
- 材质：8个（身体/脸/眼/头发/眉毛/尾巴）

**转换流程**
```
.pmx
  → Blender 5.1 + mmd_tools 插件导入
  → 运行 convert_mmd_materials.py（mmd_shader → Principled BSDF）
  → File → Export → glTF 2.0 (.glb)
  → Godot 4 拖入，贴图正常显示
```

**关键文件**
- `plugins/Godot/TokaiTeio/convert_mmd_materials.py` — 材质转换脚本
- `plugins/Godot/TokaiTeio/models/` — 导出的 .glb 模型
