1. 启动流程

flowchart TD
    A([python main.py]) --> B[加载 .env 环境变量]
    B --> C[load_config 读取配置]
    C --> D[asyncio 预加载嵌入模型\nBAAI/bge-small-zh-v1.5]
    D --> E[创建 QApplication\nFusion 主题 / 微软雅黑]
    E --> F[创建 AIAgentApp 主窗口]
    F --> G[ImprovedAIAgent 初始化]

    G --> G1[ImprovedMemorySystem\nFAISS + SQLite]
    G --> G2[ToolManager\n天气 / 搜索 / 音乐]
    G --> G3[TTS 初始化\nGPT-SoVITS 或 Azure]
    G --> G4[TextQueueManager 启动]
    G --> G5[MultimodalProcessor 初始化\nGLM-4V-Flash]

    F --> H[安装 Qt 事件过滤器]
    H --> I[启动 HTTP 服务器\nlocalhost:5000 后台线程]
    I --> J([Qt 事件循环 app.exec_])

3. 对话处理主流程

flowchart TD
    A([用户输入]) --> B{命中响应缓存?}
    B -->|是| Z([直接返回缓存])
    B -->|否| C{相同请求处理中?}
    C -->|是| D[等待已有任务完成]
    D --> Z
    C -->|否| E{简单问题?\n_is_simple_query}

    E -->|是 快速模式| F1[仅取最近1条历史]
    E -->|否 正常模式| F2[取最近3条历史\n+ 向量知识库检索]

    F1 --> G[_get_system_prompt\n角色人设 + 游戏画面缓存]
    F2 --> G

    G --> H{multimodal_enabled?}

    H -->|是 per-message 模式| I[截图 capture_full_screen\n+ GLM-4V describe_image\n与上下文获取并行]
    I --> J[image_description 注入\n用户输入文本]
    J --> K[DeepSeek 流式调用\ndeepseek-chat]

    H -->|否| K

    K -->|stream chunk| L[TextQueueManager\n按标点切分]
    L -->|完整句子| M[TTS 合成\nGPT-SoVITS / Azure]
    M --> N([语音播放])

    K --> O[保存对话到记忆系统\n异步后台任务]
    K --> P[写入响应缓存]

    style A fill:#4a90e2,color:#fff
    style Z fill:#27ae60,color:#fff
    style N fill:#27ae60,color:#fff

4. 游戏画面后台监控（新架构）

flowchart TD
    A([start_game_monitoring\ninterval / diff_threshold\ncapture_type / title_keyword]) --> B[启动 daemon 后台线程\n_monitoring_loop]

    B --> C[_capture_for_monitoring]
    C -->|capture_type=window| C1[capture_window_by_title\nwin32gui 按标题截图]
    C -->|capture_type=full| C2[capture_full_screen\npyautogui 全屏截图]
    C1 --> D[PIL resize 压缩]
    C2 --> D

    D --> E[转灰度 numpy 数组]
    E --> F{prev_array 存在?}
    F -->|否 首次| G[直接更新]
    F -->|是| H[计算像素平均差 diff]
    H --> I{diff > threshold?}
    I -->|否 画面无变化| J[跳过 打印 diff 值]
    I -->|是 画面已变化| G

    G --> K[更新 prev_screenshot_array]
    K --> L[describe_image 调用\nGLM-4V-Flash\n80字以内描述]
    L --> M[game_state_cache 更新]

    J --> N[删除临时截图]
    M --> N
    N --> O{stop_event 触发?}
    O -->|否| P[wait interval 秒]
    P --> C
    O -->|是| Q([监控停止\ncache 清空])

    M -.->|每次对话| R[_get_system_prompt\n注入 【当前游戏画面】]
    R -.-> S[DeepSeek 自行判断\n是否在回复中引用]

    style A fill:#8e44ad,color:#fff
    style Q fill:#e74c3c,color:#fff
    style S fill:#27ae60,color:#fff
