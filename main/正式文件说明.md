# 东海帝王AI - 正式文件说明

## 📁 项目结构

### 🎯 核心程序文件

| 文件名 | 说明 |
|--------|------|
| **main.py** | 主程序入口，负责初始化应用、启动HTTP服务器和主窗口 |
| **main_window.py** | 主窗口模块，包含UI界面和用户交互逻辑 |
| **improved_ai_agent.py** | AI Agent核心模块，整合异步处理和模块化架构 |
| **async_ai_agent.py** | 异步AI代理模块，提供异步向量记忆系统 |

### 🗣️ 语音合成 (TTS) 模块

| 文件名 | 说明 |
|--------|------|
| **gpt_sovits_unified.py** | 统一的GPT-SoVITS TTS调用模块，支持Gradio和api_v2两种方式 |
| **gpt_sovits_simple.py** | 简化的GPT-SoVITS TTS调用模块，通过Gradio客户端调用 |
| **gpt_sovits_tts.py** | GPT-SoVITS TTS管理器，处理通过api_v2.py调用TTS功能 |
| **api_v2_client.py** | GPT-SoVITS API v2客户端，支持流式和非流式语音合成 |
| **text_queue_manager.py** | 流式文本队列管理器，按标点符号切分文本并按顺序发送到TTS |

### 🎤 语音识别 (ASR) 模块

| 文件名 | 说明 |
|--------|------|
| **example_realtimestt.py** | 实时语音识别集成程序，将识别到的文本发送到主程序 |
| **realtime_stt_direct.py** | RealtimeSTT直接集成模块，提供实时语音识别功能 |
| **asr_integration.py** | ASR集成模块，提供RealtimeSTT与主程序ASR功能的集成 |

### 🧠 记忆系统模块

| 文件名 | 说明 |
|--------|------|
| **improved_memory.py** | 改进的记忆系统，使用FAISS + SQLite架构 |
| **memory_lake.py** | 记忆湖系统，用于存储和检索对话历史 |

### 🔧 工具模块

| 文件名 | 说明 |
|--------|------|
| **tool_manager.py** | 工具管理模块，实现模块化的工具管理 |
| **weather.py** | 天气功能模块，处理天气相关的API调用 |
| **amap_tool.py** | 高德地图API工具类 |

### ⚙️ 配置和UI模块

| 文件名 | 说明 |
|--------|------|
| **config.py** | 配置管理模块，处理应用程序的配置加载和保存 |
| **settings_dialog.py** | 设置对话框模块（浅色主题版本） |
| **ui_dialogs.py** | UI对话框模块，包含设置、记忆系统、MCP工具等对话框 |
| **utils.py** | 工具函数模块，包含系统工具、辅助函数和通用功能 |

### 🚀 启动脚本

| 文件名 | 说明 |
|--------|------|
| **run_example.bat** | STT到主程序的启动脚本 |
| **start_all.bat** | 东海帝王AI的启动脚本 |

## 💡 使用说明

### 启动主程序
```bash
# 方法1：使用启动脚本
start_all.bat

# 方法2：直接运行
python main.py
```

### 启动STT集成程序
```bash
# 使用启动脚本
run_example.bat
```

## ⚠️ 注意事项

1. 确保所有依赖包已正确安装
2. 配置文件中的API密钥需要正确设置
3. GPT-SoVITS API需要先启动
4. ASR功能需要正确配置RealtimeSTT插件
